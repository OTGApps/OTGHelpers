# OTGHelpers

A set of class extenstions I commonly use in my applications for RubyMotion.

## Installation

Add this line to your application's Gemfile:

    gem 'OTGHelpers'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install OTGHelpers
